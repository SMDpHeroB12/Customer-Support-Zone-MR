# React- Mission Restart Assignment-02 Q/A

## What is JSX, and why is it used?

<br>Answer: JSX Means javaScript XML. It is used for creating different components.

## What is the difference between State and Props?

<br>Answer: A parent component passes to a child component through "Props" or Properties. But cannot be changed by the child component.<br>
On the other hands "State" is to manage dynamic data inside a component.

## What is the useState hook, and how does it work?

<br>Answer: useState hook is a React Hook that let me add state to a function component.

## How can you share state between components in React?

<br>Answer: From Parent to child Pass it down as Props.

## How is event handling done in React?

<br>Answer: In React, event handling is done by attaching functions to camelCase events, using arrow functions to pass arguments, and e.preventDefault() to stop default behavior.
